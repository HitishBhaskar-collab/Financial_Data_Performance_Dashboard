# Financial_Data_Performance_Dashboard
This repository contains a Power BI dashboard designed to analyze financial performance across multiple business dimensions. The report highlights key metrics such as total sales, profit, profit margin, total cost of goods sold (COGS), and total discounts to support data-driven decision-making in finance and business operations.

# Features
KPI cards for Total Sales, Total Profit, Profit Margin %, Total Discounts, and Total COGS
Sales comparison across countries
Monthly sales trend visualization
Gross Sales vs Discounts performance scatter plot
Product-wise sales segmented by discount bands
Interactive slicers for Date, Country, Product, Segment, and Discount Band

# Overview
This dashboard provides a unified view of business financial performance, helping users track profitability, identify high-performing products and markets, evaluate discounts impact, and monitor seasonal sales trends. It is a practical project for financial analysts and BI developers working on data visualization and decision support.
